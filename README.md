# TaskFlow - Менеджер задач

TaskFlow - это веб-приложение для управления персональными и командными задачами. Проект помогает пользователям организовать рабочий процесс, устанавливать приоритеты и отслеживать прогресс. Основные функции включают создание задач, установку сроков, категоризацию и совместную работу. Приложение построено на современном стеке технологий с использованием React и Node.js. Цель проекта - предоставить интуитивно понятный инструмент для повышения продуктивности.

## Sprint Review

### Завершенные задачи:
1. **Настроить аутентификацию пользователей** (#1) - реализована система регистрации и входа
2. **Реализовать создание задач** (#2) - добавлен интерфейс создания и базового редактирования задач

### Доставлено по итогу спринта:
- Рабочая система аутентификации с JWT-токенами
- Базовый CRUD для задач
- Главная страница приложения

### Не завершенные задачи:
- **Добавить систему уведомлений** (#5) - не успели реализовать email-оповещения
